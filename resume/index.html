Work

NewSpring 03/2015 — 11/2015

https://newspring.cc

NewSpring is a church in South Carolina with 19 campuses. I worked on several different projects alongside a team of 8 other developers. I worked on a new Meteor framework called Apollos which will be used to replace their existing website. This included a sophisticated syncing application to sync data between Apollos and Rock (their internal management system). I helped migrate their existing web assets (10 different sites) from Rackspace to Amazon Web Services. I built a tool called Meteor Launch which allows you to build a Meteor app for iOS and Android, then deploy it to TestFlight, the iTunes Store, Hockey, and the Google Play Store, with only one command. I also built a tool called Meteor Package which will build a Meteor app for iOS, Android, Chrome, Windows, Mac OS X, Linux, and Fire OS.
